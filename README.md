# Data Analytics Tool

## Description
A simple Data Analytics Tool built using HTML, CSS, JavaScript, and optional Python backend for uploading, cleaning, analyzing, and visualizing data.

## Features
- CSV file upload
- Data preview table
- Missing value detection
- Data cleaning options
- Summary statistics
- Column filtering
- Sorting data
- Correlation matrix
- Interactive charts generation
- Export cleaned data/download report

## Technologies Used
- HTML
- CSS
- JavaScript
- Python (optional)

## How to Run
Open `index.html` in your browser.
